# Informacje ogólne o projekcie

## Dane podstawowe

Nazwa projektu: Cardentis  
Typ projektu: Projekt indywidualny  
Rodzaj gry: Gra webowa – karciana gra RPG z elementami strategicznymi  
Platforma: Przeglądarka internetowa (SPA)

## Wykorzystywane technologie

- Backend / logika gry: C# – Blazor Server (ASP.NET Core)
- Frontend: Blazor Server (interfejs webowy, SPA)
- Komunikacja w czasie rzeczywistym: SignalR
- Baza danych: SQLite (Entity Framework Core)

## Opis koncepcji gry

Cardentis jest karcianą grą RPG, w której gracz wciela się w postać rozwijaną poprzez:

- zdobywanie i kolekcjonowanie kart,
- udział w turowych walkach,
- zarządzanie własną talią kart,
- rozwój postaci oraz posiadanych kart.

Rozgrywka opiera się na podejmowaniu decyzji strategicznych, planowaniu ruchów oraz odpowiednim doborze kart do aktualnej sytuacji w grze.

Gra wykorzystuje turowy system rozgrywki, w którym gracze planują swoje ruchy, a następnie następuje wspólne rozliczenie rundy.

Podstawowym trybem gry jest multiplayer 1v1, w którym dwóch graczy rywalizuje ze sobą w czasie rzeczywistym.  
Opcjonalnie możliwe jest zaimplementowanie trybu singleplayer jako rozszerzenia projektu.
